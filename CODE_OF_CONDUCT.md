# BitcoinBT Code of Conduct

## Our Commitment

We are committed to providing a welcoming, professional, and constructive environment for all participants in the BitcoinBT community.

We encourage respectful collaboration among developers, miners, node operators, researchers, and community members regardless of experience level, nationality, language, or background.

Our goal is to support open technical discussion and the long-term development of the BitcoinBT ecosystem.

---

## Expected Behavior

Examples of behavior that contribute to a positive environment include:

* Being respectful and professional
* Providing constructive technical feedback
* Discussing ideas and code rather than attacking individuals
* Accepting differing viewpoints and experiences
* Helping new users and contributors when possible
* Reporting issues responsibly and accurately
* Focusing on facts, evidence, and technical discussion

---

## Unacceptable Behavior

Examples of unacceptable behavior include:

* Personal attacks or harassment
* Threats or intimidation
* Discriminatory or hateful language
* Deliberate misinformation
* Spam, trolling, or disruptive behavior
* Publishing private information without permission
* Repeated bad-faith accusations without evidence

Technical criticism and disagreement are welcome. Personal hostility is not.

---

## Technical Discussions

BitcoinBT is an open-source project.

Community members are encouraged to:

* Review source code
* Verify builds independently
* Report bugs and vulnerabilities responsibly
* Challenge technical assumptions with evidence

Constructive skepticism is valuable and helps improve the project.

---

## Reporting Problems

If you experience behavior that violates this Code of Conduct, please contact the project maintainers:

Email: [info@bitcoinbt.xyz](mailto:info@bitcoinbt.xyz)

Reports will be reviewed and handled in good faith.

---

## Enforcement

Project maintainers may remove comments, discussions, issues, pull requests, or other content that violates this Code of Conduct.

Repeated or severe violations may result in restrictions on participation within project-managed platforms.

---

## Scope

This Code of Conduct applies to:

* GitHub Issues
* GitHub Discussions
* Pull Requests
* Project communication channels
* Official BitcoinBT community spaces
